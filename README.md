# Ji Yang — UX/UI Product Designer Portfolio

A minimal, editorial portfolio site showcasing UX/UI product design work grounded in research and reasoning.

**Live site:** [mickie0917.github.io/Portfolio](https://mickie0917.github.io/Portfolio) *(update if hosted elsewhere)*

## Case Studies

| # | Project | Focus |
|---|---------|-------|
| 01 | **Binaural Sounds** | Music-based emotional tuning service designed from 15 in-depth interviews and the KANO model |
| 02 | **Plowguys** | Crew member app redesign for snow removal workers — built for thick gloves, harsh weather, and field conditions |
| 03 | **Makahealth** | B2C patient portal unifying scattered health records into one actionable summary, shipped in 12 weeks |

Each case study follows the **Decisive One Page** structure:
**Problem → Hypothesis → Evidence → Design Decision (with trade-offs) → Outcome**

## Features

- Clean, grid-paper aesthetic with scroll-reveal animations
- Bilingual support (English / Korean) via in-page toggle
- Fully responsive — mobile-first layout
- Terminal-style status bar

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- Google Fonts (Space Grotesk, IBM Plex Mono)
- No build tools required — open `index.html` to view

## Getting Started

```bash
git clone https://github.com/mickie0917/Portfolio.git
cd Portfolio
open index.html
```
